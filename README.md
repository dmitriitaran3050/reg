# Минилендинг регистрации на повторы эфиров

Статичный мобильный лендинг для регистрации на повторы эфиров через GitHub Pages.

## Файлы

- `index.html` — разметка страницы и расписание повторов.
- `styles.css` — мобильные стили.
- `script.js` — ссылки регистрации и цели Яндекс.Метрики.
- `assets/host.jpg` — фото ведущей для первого экрана.
- `assets/og.svg` — отдельная картинка для предпросмотра при репосте.

## Что заменить перед публикацией

В `script.js` замените ссылки в `CONFIG.links`:

- `max`
- `telegram`
- `vk`

Сейчас стоят технические плейсхолдеры вида `https://t.me/your_bot?start=money`.
